================================================================================
PROBLEM: APB Master–Slave RTL Debug and Integration
================================================================================
**Task:** Debug and complete an AMBA APB-based RTL design.

**Your Task:** Fix and complete the RTL implementation in the rtl/ directory so that the APB system works correctly and passes the provided testbench.

**DUT:** apb_top (in apb_rtl/sourcws/apb_top.sv) — this module connects apb_master and apb_slave internally.
Instantiate and connect the master and slave inside apb_top. 

**Requirements**
1. Complete Top-Level Integration
  •	Implement the apb_top module.
  •	Instantiate:
    •	apb_master
    •	apb_slave

Connect all APB bus signals between the master and slave:
`PADDR`
`PWDATA`
`PRDATA`
`PWRITE`
`PSELx`
`PENABLE`
`PREADY`
`PSLVERR`

**2. Fix APB Master FSM Behavior**
The APB master must implement the standard APB protocol state machine:

**States:**
  •	IDLE
  •	SETUP
  •	ACCESS

**Expected behavior:**
1.	IDLE
  •	PSELx = 0
  •	PENABLE = 0
  •	Wait for TRANSFER = 1.

2.	SETUP
  •	PSELx = 1
  •	PENABLE = 0
  •	Address (PADDR) and control (PWRITE) become valid.

3.	ACCESS
  •	PSELx = 1
  •	PENABLE = 1
  •	Transaction completes when PREADY = 1.

**3. Implement Read and Write Transactions**
**Write Operation**
  •	WRITE_IN = 1
  •	PWRITE = 1
  •	Data from WDATA must be driven on PWDATA
  •	Address must be driven on PADDR

**Read Operation**
  •	WRITE_IN = 0
  •	PWRITE = 0
  •	Data from PRDATA must be captured into RDATA.

**4. Slave Register File Behavior**
The APB slave contains a register file with:
  •	REG_NUM registers
  •	Each register width = DATA_WIDTH

Registers are word aligned, and the register index is derived from:
PADDR[ADDR_WIDTH-1:2]
**Write condition**
  •	PSELx = 1
  •	PENABLE = 1
  •	PWRITE = 1	
Write PWDATA to the selected register.
**Read condition**
  •	PSELx = 1
  •	PENABLE = 1
  •	PWRITE = 0
Return the selected register value on PRDATA.

**5. Reset Behavior**
When PRESET_n = 0:
  •	All registers must reset to 0
  •	PRDATA = 0
  •	PSLVERR = 0
  •	PREADY = 1

**6. Testbench Compatibility**
A verification testbench is provided in the tests directory.
Your corrected RTL must:
  •	Compile successfully
  •	Pass all testcases in the provided testbench
  •	Produce correct read/write behavior for the APB register file
The simulation should complete without errors and all testcases must pass.
See docs/Specification.md for detailed APB protocol behavior and signal descriptions.
